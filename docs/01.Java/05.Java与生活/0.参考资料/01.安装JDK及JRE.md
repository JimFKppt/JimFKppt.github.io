# jdk安装方法：以下可以用来复制粘贴

## 1.新建系统变量：JAVA_HOME

路径取jdk路径

**E:\xxx\jdk1.8**

## 2.在系统变量Path后面追加

%JAVA_HOME%\bin

%JAVA_HOME%\jre\bin

## 3. 添加classpath

CLASSPATH

.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar
