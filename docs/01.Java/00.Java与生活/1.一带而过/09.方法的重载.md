# 方法的重载

## 理念

同一个方法可以传入不同类型不同个数的参数，实现方法名的统一。

> 这并不意味着只用写一个方法。

## 核心

- 方法名**相同**
- 参数类型或者参数个数**不相同**

```Java
// 两个int类型参数
public static int sum(int x, int y) {
    return x + y;
}

// 两个double类型参数
public static double sum(double x, double y) {
    return x + y;
}

// 三个int类型参数
public static int sum(int x, int y, int z) {
    return x + y + z;
```
