# toString()

## 什么是“toString()"?

https://jabingp.github.io/javaRewriteToString.html

`toString()` 是 Object 类的一个方法。通过重写 `toString()` 在我们自己的类中使用其功能。

功能：

> 打印对象，为了让大家的输出好看。

就是toString 可能是一个系统性能的忧患，但是~~~

初学者，爽就完事儿了！！！

## IDEA快捷方式

- 在类中，`Alt+Insert`
- 选择 toString()
- 完事儿

## 尽管IDEA已经非常便利了

但是，还是不够逼格。。。

那怎样才能B格到位呢？

Lombok ---> 一行注解 `@ToString`，完事儿。

如果要同时使用 toString()、equals()、hashCode() 三个，只要一个 `@Data` 就行了。
