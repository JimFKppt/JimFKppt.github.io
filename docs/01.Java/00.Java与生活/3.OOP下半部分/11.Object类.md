# Object类

Object 类是Java中所有类的父类，包括你本人定义的类，它还有一个霸气的名字————**超类**。

例如经常使用的 `toString()` 就是 Object 类中的一个方法。

​	所以 `toString()` 方法上边都需要加一个 `@Overide` 注解来**重写** Object 类中的 `toString()` 方法。
