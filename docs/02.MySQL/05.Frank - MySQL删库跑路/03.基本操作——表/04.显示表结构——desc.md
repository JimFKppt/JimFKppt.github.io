# 显示表结构

```sql
desc <TableName>;
```

or

```sql
describe <TableName>;
```

Example: 

```sql
mysql> desc students;
+---------+--------------+------+-----+----------+----------------+
| Field   | Type         | Null | Key | Default  | Extra          |
+---------+--------------+------+-----+----------+----------------+
| id      | int(11)      | NO   | PRI | NULL     | auto_increment |
| name    | varchar(30)  | NO   |     | NULL     |                |
| age     | int(11)      | YES  |     | NULL     |                |
| phone   | varchar(20)  | YES  |     | NULL     |                |
| address | varchar(100) | YES  |     | 暂时未知  |                |
+---------+--------------+------+-----+----------+----------------+
```
