# 引用数据库和查看数据库中的表

## 引用数据库

```sql
use <DatabaseName>;
```

Example:

```sql
use students;
```

为什么要引用数据库？

​	就像京东指定哪个仓库发货一样。

## 查看数据库中的表

```sql
show tables;
```

Example:

```sql
mysql> show tables;
Empty set (0.00 sec)
```
