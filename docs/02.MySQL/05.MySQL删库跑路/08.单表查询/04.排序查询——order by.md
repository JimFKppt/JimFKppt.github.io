# 排序查询——order by

`order by` 排序

—— `asc` 代表**升序**

—— `desc` 代表**降序**

```mysql
+----+------+------------+---------------+
| id | name | math_score | english_score |
+----+------+------------+---------------+
|  1 | Tom  |         20 |            99 |
|  2 | Jim  |         59 |            77 |
|  3 | Anna |         77 |            59 |
|  4 | 张三 |         89 |            89 |
|  5 | 李四 |         99 |            20 |
+----+------+------------+---------------+
```

## 升序

```mysql
select id,name,english_score from student order by english_score asc;
```

```mysql
+----+------+---------------+
| id | name | english_score |
+----+------+---------------+
|  5 | 李四 |            20 |
|  3 | Anna |            59 |
|  2 | Jim  |            77 |
|  4 | 张三 |            89 |
|  1 | Tom  |            99 |
+----+------+---------------+
```

## 降序

```mysql
select id,name,english_score from student order by english_score desc;
```

```mysql
+----+------+---------------+
| id | name | english_score |
+----+------+---------------+
|  1 | Tom  |            99 |
|  4 | 张三 |            89 |
|  2 | Jim  |            77 |
|  3 | Anna |            59 |
|  5 | 李四 |            20 |
+----+------+---------------+
```
