# 模糊查询——like

`where .. like`

—— `%` 符号代表多个字符

—— `_` 符号代表一个字符

```mysql
+----+------+
| id | name |
+----+------+
| 1  | Tom  |
| 2  | Jim  |
| 3  | Anna |
| 4  | 张三 |
| 5  | 李四 |
+----+------+
```

## `%` 百分号

```mysql
select name from student where name like 'A%';
```

```mysql
+------+
| name |
+------+
| Anna |
+------+
```

## `_` 下划线

```mysql
select name from student where name like '张_';
```

```mysql
+------+
| name |
+------+
| 张三 |
+------+
```

```mysql
select name from student where name like 'An__';
```

```mysql
+------+
| name |
+------+
| Anna |
+------+
```
