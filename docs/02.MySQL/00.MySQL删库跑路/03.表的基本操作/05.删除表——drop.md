# 删除表

```mysql
drop table <TableName>;
```

亦可同时删除多个表，并且可以加上 `if exists` 语句。

Example: 

```mysql
mysql> drop table if exists teachers, students;
Query OK, 0 rows affected (0.01 sec)
```
