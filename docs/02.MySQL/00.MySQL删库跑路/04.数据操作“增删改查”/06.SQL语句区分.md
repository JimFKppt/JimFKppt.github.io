# DDL

Data Difintion Language，数据库定义语言

`create` 、`drop` 、`alter`、`show`

# DML

Data Manipulation Language，数据操作语言

`insert` 、`delete` 、`update`、`select`

# DCL

Data Control Language，数据库控制语言
