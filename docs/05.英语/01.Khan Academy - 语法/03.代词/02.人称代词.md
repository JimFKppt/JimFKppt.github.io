# 人称代词

## 1st 第一人称 ---> me

| Singular              | Plural                  |
| --------------------- | ----------------------- |
| I、me、<br />my、mine | we、us、<br />our、ours |

## 2nd 第二人称 ---> you

| Singular               | Plural                 |
| ---------------------- | ---------------------- |
| you、your、<br />yours | you、your、<br />yours |

## 3rd 第三人称 ---> people or things

| Singular                                          | Plural                          |
| ------------------------------------------------- | ------------------------------- |
| she、her、hers、<br />he、him、his、<br />it、its | they、them、<br />their、theirs |
